# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Development Commands

**Development Workflow:**

- `npm run dev` - Start Tailwind CSS in watch mode (development)
- `npm run build` - Build minified CSS for production

**File Structure:**

- CSS Source: `./assets/css/src.css` (input for Tailwind)
- CSS Output: `./assets/css/main.css` (generated by Tailwind)
- HTML Files: `index.html`, `front-page.html` (primary templates)
- Assets: `./  /` directory (contains CSS and other static assets)

**Tailwind Configuration:**

- Custom screens defined: `tab-sm`, `tab-lg`, `md-screen`, `lg-screen`, `xl-screen`
- Brand colors: `brand` (#E63312), `ink` (#000000), `surface` (#FFFFFF), `accent` (#FFA800), `dark` (#512C36)
- Custom typography: `banner-heading`, `section-heading`, `section-heading-h3`, `body` with fluid clamp() values
- Custom spacing: `section` (clamp-based), `gutter` (1.5rem)
- UI elements: `card` border-radius, `pill` border-radius, `card` and `header` box-shadow
- Animations: `marquee`, `marquee2`, `spin-slow` with corresponding keyframes

**Important Notes:**

- The `src/` directory referenced in tailwind.config.js does not exist - Tailwind will only process the explicitly listed HTML files
- CSS modifications should be made to `./assets/css/src.css`
- Generated CSS output is `./assets/css/main.css` - do not edit this file directly
- When adding new HTML files, add them to the `content` array in tailwind.config.js
